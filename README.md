# Online Sınav Sistemi

Bu proje Marmara Üniversitesi Bilgisayar Mühendisliği Yazılım Mühendisliği dersi kapsamında geliştirilmiştir.

## Özellikler
- **Öğretmen Paneli:** Sınav oluşturma, süre belirleme ve sonuçları görüntüleme.
- **Öğrenci Paneli:** Mevcut sınavlara giriş, canlı süre takibi ve anlık puan hesaplama.
- **Mimari:** Flask (Python) tabanlı MVC yapısı.

## Kurulum
1. `pip install -r requirements.txt`
2. `python app.py`
