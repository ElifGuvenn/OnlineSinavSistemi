Proje Adı

ExamFlow - Online Sınav Yönetim Sistemi

Proje Tanımı

ExamFlow, öğretmenlerin kolayca çevrimiçi sınavlar oluşturabildiği, öğrencilerin ise kendi eğitim seviyelerine uygun sınavlara girip anında detaylı sonuç analizi alabildiği modern bir çevrimiçi sınav platformudur.

Özellikler

Dinamik Sınav Oluşturma ve zaman kısıtlamaları (Time Windows).

Öğrenci okul düzeyi ve sınıfına göre otomatik sınav eşleştirme.

Tam ekran "Odak Modu" ve geri sayım sayacı ile sınav uygulaması.

Sınav sonrası anında doğru/yanlış analizi ve kapsamlı öğretmen paneli.

Kullanılan Teknolojiler

Backend: Python, Flask Framework

Veritabanı: SQLite, Flask-SQLAlchemy (ORM)

Frontend: HTML5, CSS3, Vanilla JS

Kurulum Adımları

git clone https://github.com/umayyklc/OnlineSinavSistemi komutu ile projeyi indirin.

pip install Flask Flask-SQLAlchemy Flask-Login bağımlılıklarını kurun.

Terminalde projenin ana dizinine geçerek python app.py komutunu çalıştırın.

Kullanım

Uygulama çalıştıktan sonra tarayıcınızdan http://127.0.0.1:5000 adresine gidiniz. Öğretmen girişi yapmak için sisteme kayıt olabilir veya öğrenci modülünü test etmek için yeni bir öğrenci profili oluşturup okul seviyenizi belirleyebilirsiniz.
